5Finishing

Only turn cutting marks on if you're printing onto paper larger than the file's page size — the marks show where to trim it down to size.

In your printer's settings, set Scale to 100% / Actual size so it prints true to size. Don't use "Fit to page", "Scale to fit", "Shrink to fit", or any stretch/fill option — those will throw the marks off.

6Export
Builds one print-ready file with every signature combined. Manual duplex splits it into a fronts file and a backs file. A separate cover exports as its own file too — up to 3 files total (cover + fronts + backs).

Tip: if your printer does have a duplex mode, set it to flip on the short edge for booklet jobs — most "flip on long edge" defaults will print the backs upside down.
